# Ionic-5-Stars-Worker

Experimental Ionic app for searching and assessing freelancers.

### Product Purpose

Helping you find well evaluated workers such as electricians, plumbers and housekeepers.

### Features

* Search by criteria
* Customer comments
* Five-star rank
