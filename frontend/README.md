# Frontend

Web App for Android and iOS devices.

### Requirements

* [Node.js][1]
* [Ionic][2]
* [Cordova][3]

### Running

Start the [backend][4] than:

```shell
npm install
ionic serve
```

Open it on your browser:

http://localhost:8100

[2]:https://ionicframework.com/
[3]:https://cordova.apache.org/
